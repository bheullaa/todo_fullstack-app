# Full Stack Todo App (CRUD)

A full stack Todo application built using HTML, CSS, JavaScript, Express.js, and MongoDB.

## Features
- Add new todos
- View all todos
- Edit existing todos
- Mark todos as completed
- Delete todos

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub

## Project Structure
backend/
├── routes/
├── models/
├── server.js
frontend/
├── index.html
├── style.css
├── script.js


## How to Run Locally
1. Clone the repository
2. Run backend:

cd backend
npm install
node server.js

3. Open `frontend/index.html` using Live Server
